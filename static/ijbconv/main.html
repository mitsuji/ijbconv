ijbconv web

To use like api with curl

curl -X POST http://example.com/ijbconv/bt -F "binary=@{filepath}"
curl -X POST http://example.com/ijbconv/tb --data-urlencode "text@{filepath}"

curl -X POST http://example.com/ijbconv/bt -F "binary=@data/test1.bin"
curl -X POST http://example.com/ijbconv/tb --data-urlencode "text@data/test1.txt"
