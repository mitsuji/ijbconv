Kana Output:

To use as HTTP API with curl

curl -X POST http://example.com/ijbconv/bt -F "binary=@{filepath}"
curl -X POST http://example.com/ijbconv/bt -F "binary=@{filepath}" -F "kana-decode={kana-decode-type}"
curl -X POST http://example.com/ijbconv/tb --data-urlencode "text@{filepath}"

for example:
curl -X POST http://example.com/ijbconv/bt -F "binary=@data/test1.bin"
curl -X POST http://example.com/ijbconv/bt -F "binary=@data/test1.bin" -F "kana-decode=katakana-half"
curl -X POST http://example.com/ijbconv/bt -F "binary=@data/test1.bin" -F "kana-decode=katakana-full"
curl -X POST http://example.com/ijbconv/bt -F "binary=@data/test1.bin" -F "kana-decode=hiragana-full"
curl -X POST http://example.com/ijbconv/tb --data-urlencode "text@data/test1.txt"
