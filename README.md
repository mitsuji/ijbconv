# ijbconv


## build

$ stack build


## install

$ stack install


## execute

### show code text of binary file
$ ~/.local/bin/ijbconv-exe < code.bin

### save code text of binary file
$ ~/.local/bin/ijbconv-exe < code.bin > code.txt

### save code binary of text file
$ ~/.local/bin/ijbconv-exe bt < code.txt > code.bin
